Roadmap

What’s next for the universal proof layer — built openly, privacy-first, and edge-native. The goal: make timestamp proofs an obvious default for every AI agent and developer workflow.

Current: v0.1 (Public Beta) Infra: Cloudflare Workers + KV Focus: DX & reliability

v0.2 — Developer Experience

  • JavaScript SDK (browser & Node): createFromText/File/Hash, verify
  • PDF certificate (downloadable proof bundle)
  • Verify UI: “Copy as cURL” & “Copy JSON”
  • Docs refresh: limits, examples, FAQ

Goal: integrate in minutes; human-friendly proof artifacts.

Read the Docs

v1.0 — Productization

  • API keys & usage quotas (Free/Pro/Team)
  • Dashboard (analytics, CSV export, key rotation)
  • Webhooks (success/failure callbacks)
  • Billing (Stripe) — predictable, no gas

Goal: reliable operations at scale with predictable cost.

See Pricing

v2.0 — Validation Layer

  • ProofChain (light distributed validation/anchoring)
  • SDKs: Python & Go
  • Advanced verify (proof bundles, offline verification)

Goal: extra assurance without sacrificing speed or privacy.

Backlog / Ideas

  • Signed agents: per-agent keys & attestations
  • CLI for CI/CD timestamping
  • S3 / GCS integration (hash on upload)
  • Org features: roles, audit export, SIEM
  • TSA compatibility (RFC 3161 – research)

Shaped by real usage & partners.

Principles

  • Privacy-first — only hashes, never content
  • Open & simple — minimal protocol, verifiable by anyone
  • Edge-native — global low latency, automatic scale
  • Predictable cost — no gas, no chain fees
